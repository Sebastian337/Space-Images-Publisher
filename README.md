# 1. Описание проекта Space Images Publisher

Набор скриптов для скачивания космических фотографий из различных источников (NASA, SpaceX) и их автоматической публикации в Telegram-канал.

## 2. Требования к окружению

- ОС: Linux, macOS, Windows
- Python: версия 3.8 или выше
- Доступ к интернету для запросов к API
- Токен NASA API (опционально, есть демо-режим)
- Telegram бот и канал

## 3. Как установить

### Установите зависимости:
```bash
pip install -r requirements.txt
```

### Настройте переменные окружения:
Создайте файл .env в корне проекта и добавьте:

```txt
# Ключ NASA API (получите на https://api.nasa.gov)
NASA_API_KEY=ваш_ключ_nasa

# Токен Telegram бота (создайте через @BotFather)
BOT_TOKEN=токен_вашего_бота

# ID Telegram канала (добавьте бота администратором)
CHAT_ID=id_вашего_канала

# Задержка между публикациями в часах (по умолчанию 4)
DELAY_HOURS=4
```

## 4. Использование скриптов

Скачивание фотографий:

```bash
# Скачать фото последнего запуска SpaceX
python fetch_spacex_images.py

# Скачать фото конкретного запуска SpaceX
python fetch_spacex_images.py --id 5eb87d47ffd86e000604b38a

# Скачать фото дня от NASA (APOD)
python fetch_nasa_apod.py

# Скачать фото Земли от NASA (EPIC)
python fetch_nasa_epic.py
```

Публикация в Telegram:

```bash
# Опубликовать одну случайную фотографию
python publish_photo.py

# Автоматическая публикация всех фотографий
python publish_all_photos.py
```

## 5. Пример запуска и проверка результата

```bash
# Скачиваем фото SpaceX
$ python fetch_spacex_images.py --id 5eb87d47ffd86e000604b38a

# Публикуем случайное фото
$ python publish_photo.py
# В Telegram-канале появится случайная космическая фотография

# Запускаем автоматическую публикацию
$ python publish_all_photos.py
# Фото будут публиковаться каждые 4 часа
```

## Примечание:

* Фотографии сохраняются в папку images/ в соответствующих подпапках
* NASA API может быть недоступен в некоторых регионах. В этом случае скрипты автоматически используют демо-ключ (DEMO_KEY) с ограниченным количеством запросов
* Для скачивания фотографий выполняя python fetch_nasa_apod.py необходимо включить VPN
* SpaceX API не всегда содержит фотографии для последнего запуска. Рекомендуется использовать конкретные ID запусков
* Для автоматической публикации частота задается переменной DELAY_HOURS в .env (по умолчанию 4 часа)
* Скрипт publish_all_photos.py публикует все фото по кругу в случайном порядке
